Quick start and integration notes for using Prompteus with Google Cloud.

Quick Start

  1. Connect Prompteus to your Google Cloud project.
  2. Configure IAM roles for Vertex AI, Cloud Run, and BigQuery.
  3. Create your first workflow in the Prompteus UI.
  4. Add tasks that call Vertex AI models for summarization, classification, or Q&A.
  5. Publish the workflow and invite teams to run it.

Example: Vertex AI Integration

Below is a simplified pseudo-flow of how Prompteus calls Vertex AI within a workflow:

1. Receive request from Prompteus workflow
2. Build prompt with context, user input, and system rules
3. Call Vertex AI text model endpoint
4. Validate and post-process the response
5. Route result to the next step (human review, storage, or another model)

Logging & Observability

  • Workflow executions can be logged into BigQuery for analysis.
  • Create dashboards to track model usage, latency, and failure rates.
  • Export audit logs for compliance and reporting purposes.
